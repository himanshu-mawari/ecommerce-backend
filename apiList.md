# E-COMMERCE APIs

# authRouter

- POST api/user/signup
- POST api/user/login
- POST api/user/logout

# productRouter

- POST /api/products/add
- DELETE /api/products/remove
- GET /api/products/:id
- GET /api/products/list

# cartRouter

- POST api/cart/add
- POST api/cart/update/:productId
- DELETE api/cart/remove/:productId
- GET api/cart/
